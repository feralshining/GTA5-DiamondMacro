### 💡 How To Use

- `F2 : CEO 되기`
- `F3 : 바이커 되기`
- `F4 : 유틸 창 오픈`
- `F5 : 불 샤크 투하`
- `F6 : 유령 연맹`
- `F7 : 관계자 매수`
- `Ctrl + Num0 : 패시브 모드`
- `Ctrl + Num1 : 잠수 방지 => 잠수 이후 아무 키 입력 시 모드 해제`
---
### ⚙️ 부가 기능
- `채팅 매크로 기능`
- `1인 공개세션`
- `GTA5 강제 종료`
